Log a decision to `docs/DECISION_LOG.md`.

Ask for the following details (or extract from context if already discussed):
1. **Decision title** — short summary
2. **Context** — why this decision was needed
3. **Decision** — what was decided
4. **Alternatives considered** — what else was considered and why it was rejected
5. **Consequences** — expected impact and trade-offs

Then:
1. Read `docs/DECISION_LOG.md` to find the last decision number
2. Append a new entry with the next number (D002, D003, etc.)
3. Set today's date and your agent ID
4. Commit with message: `[docs] add: decision D[NNN] — [title]`
